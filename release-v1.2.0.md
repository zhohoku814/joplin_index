# v1.2.0 Release

## 中文

这是 `Rich TOC Top / 顶部目录增强` 的稳定版发布。

### 主要内容

- 在当前笔记顶部生成 / 刷新目录
- 兼容 Joplin 富文本编辑
- 提供侧边目录面板
- 长目录支持滚动
- 支持层级过滤：
  - 只显示顶级目录
  - 仅显示 1-2 级
  - 显示全部层级
- 点击侧边目录项可跳转到对应标题
- 使用 GPL-3.0-or-later 开源协议

### 安装

下载并安装附件中的 `com.openclaw.richTocTop.jpl`：

`工具 -> 选项 -> 插件 -> 从文件安装`

## English

This is the stable release of `Rich TOC Top / 顶部目录增强`.

### Highlights

- Generate / refresh a TOC at the top of the current note
- Compatible with Joplin rich-text editing
- Sidebar outline panel
- Scrollable outline for long notes
- Level filters:
  - Top level only
  - Levels 1-2 only
  - Show all levels
- Click outline items to jump to headings
- Licensed under GPL-3.0-or-later

### Install

Download the attached `com.openclaw.richTocTop.jpl` and install it via:

`Tools -> Options -> Plugins -> Install from file`
