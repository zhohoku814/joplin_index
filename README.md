# Joplin Rich TOC Top / Joplin 顶部目录增强插件

在当前 Joplin 笔记顶部一键生成或刷新目录，并提供支持滚动与层级过滤的侧边目录面板，兼容富文本编辑。

Generate or refresh a table of contents at the top of the current Joplin note, with a sidebar outline that supports scrolling and level filtering, while remaining friendly to rich-text editing.

## 功能 Features

- 在当前笔记顶部生成 / 刷新目录  
  Generate / refresh a TOC at the top of the current note.
- 兼容 Joplin 富文本编辑场景  
  Works with Joplin rich-text editing workflows.
- 提供侧边目录面板  
  Provides a sidebar outline panel.
- 侧边目录支持长列表滚动  
  Supports scrolling for long outlines.
- 支持目录层级过滤：  
  Supports outline level filtering:
  - 只显示顶级目录 / Top level only
  - 仅显示 1-2 级 / Levels 1-2 only
  - 显示全部层级 / Show all levels
- 点击侧边目录项可跳转到对应标题  
  Click sidebar items to jump to headings.
- 可移除插件生成的顶部目录  
  Can remove the TOC generated by the plugin.

## 安装 Installation

### 直接安装打包文件 / Install from packaged file

在 Joplin 中打开：  
Open Joplin and go to:

`工具 -> 选项 -> 插件 -> 从文件安装`  
`Tools -> Options -> Plugins -> Install from file`

选择仓库中的：  
Select the packaged file in this repository:

- `com.openclaw.richTocTop.jpl`

然后重启 Joplin。  
Then restart Joplin.

## 使用说明 Usage

### 生成顶部目录 / Generate top TOC

- 在笔记中使用标题 1 / 标题 2 / 标题 3 等标题层级  
  Use Heading 1 / Heading 2 / Heading 3 and other heading levels in the note.
- 点击编辑器工具栏中的目录按钮，或在工具菜单中执行“生成/刷新顶部目录”  
  Click the TOC button in the editor toolbar, or run “Generate/Refresh Top TOC” from the Tools menu.

### 侧边目录 / Sidebar outline

- 通过工具菜单或按钮显示 / 隐藏侧边目录  
  Show or hide the sidebar outline from the Tools menu or toolbar button.
- 目录过长时会自动滚动  
  The outline becomes scrollable when it is long.
- 可在侧边栏中切换显示范围：  
  You can switch the display range in the sidebar:
  - 只显示顶级目录 / Top level only
  - 仅显示 1-2 级 / Levels 1-2 only
  - 显示全部 / Show all

> 说明 / Note: 这里的“顶级”是相对当前笔记中最高层标题计算的，不会强制写死为 Markdown 的 `#`。  
> “Top level” is calculated relative to the highest heading level present in the current note, rather than being hard-coded to Markdown `#`.

## 开发 Development

```bash
npm install
npm run dist
```

> 注意 / Note: 本项目在部分 WSL / Windows 混合环境下，官方打包脚本可能会遇到路径兼容问题。必要时可手工打包 `.jpl`。  
> In some mixed WSL / Windows environments, the default build script may run into path compatibility issues. If needed, the `.jpl` package can be created manually.

- `src/`：源码 / Source code

- `com.openclaw.richTocTop.jpl`：可直接安装的插件包 / Installable plugin package
- `com.openclaw.richTocTop.json`：插件清单 / Plugin manifest metadata
