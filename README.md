# Joplin Rich TOC Top

在当前 Joplin 笔记顶部一键生成或刷新目录，并提供一个更顺手的侧边目录面板，兼容富文本编辑。

## 当前功能

- 在当前笔记顶部生成/刷新目录
- 兼容 Joplin 富文本编辑场景
- 可显示侧边目录面板
- 侧边目录支持长列表滚动
- 支持目录层级过滤：
  - 只显示顶级目录
  - 仅显示 1-2 级
  - 显示全部层级
- 点击侧边目录项可跳转到对应标题
- 可移除插件生成的顶部目录

## 安装

### 方式一：直接安装打包文件

在 Joplin 中打开：

`工具 -> 选项 -> 插件 -> 从文件安装`

选择仓库中的：

- `com.openclaw.richTocTop.jpl`

然后重启 Joplin。

## 使用说明

### 生成顶部目录

- 在笔记中使用标题 1 / 标题 2 / 标题 3 等标题层级
- 点击编辑器工具栏中的目录按钮，或在工具菜单中执行“生成/刷新顶部目录”

### 侧边目录

- 通过工具菜单或按钮显示/隐藏侧边目录
- 目录过长时会自动滚动
- 可在侧边栏中切换显示范围：
  - 只显示顶级目录
  - 仅显示 1-2 级
  - 显示全部

> 说明：这里的“顶级”是相对当前笔记中最高层标题计算的，不会强制写死为 Markdown 的 `#`。

## 开发

```bash
npm install
npm run dist
```

> 注意：本项目当前在部分 WSL / Windows 混合环境下，官方打包脚本可能会遇到路径兼容问题。源码仍可正常维护，必要时可手工打包 `.jpl`。

## 仓库内容说明

- `src/`：源码
- `api/`：Joplin 插件 API 类型与适配文件
- `com.openclaw.richTocTop.jpl`：可直接安装的插件包
- `com.openclaw.richTocTop.json`：插件清单
